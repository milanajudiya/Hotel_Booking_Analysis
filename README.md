<h2> :clipboard: Data Description</h2>
Data Contains 32 features and more than 1 lakh rows, Features given in dataset are:

> * hotel = Resort hotel and city hotel
> * is_canceled = Booking Was Canceled(1) or not(0)
> * lead_time = it is the time between Booking date of hotel till arrival date
> * arrival_date_year = Year of arrival
> * arrival_date_month = Month of arrival
> * arrival_date_week_number = week number of arrival
> * arrival_date_day_of_month = Day of arrival
> * stays_in_weekend_nights = Number of weekend nights(Saturday and Sunday)
> * stays_in_week_nights = Number of week nights(Monday to Friday)
> * adults = Number of adults
> * children = Number of children
> * babies = Number of babies
> * meal = Types of Meal - categories are 'BB', 'FB', 'HB', 'SC' and 'Undefined'
> * Country = Guest/customer are from which country
> * market_segment = market segment are 'Direct', 'Corporate', 'Online TA', 'Offline TA/TO','Complementary', 'Groups', 'Undefined', 'Aviation',where TO: means "Tour Operator", TA: means "Travel Agent"
> * distribution_channel = Booking through distribution channel are 'Direct', 'Corporate', 'TA/TO', 'Undefined', 'GDS', where TO: means "Tour Operator", TA: means "Travel Agent"
> * is_repeated_guest = Guest is repeated(1) or not repeated(0)
> * previous_cancellation = Previously booking cancelled(1) nor not(0) before current booking
> * previous_bookings_not_cancelled = Previously booking not cancelled(1) nor not(0) before current booking
> * reserved_room_type = Room reserved by guest: categories of rooms :'C', 'A', 'D', 'E', 'G', 'F', 'H', 'L', 'P', 'B'
> * assigned_room_type = Room assigned to guest: categories of rooms :'C', 'A', 'D', 'E', 'G', 'F', 'I', 'B', 'H', 'P', 'L', 'K'
> * booking_changes = Number of booking changes by Guest
> * deposite_type = Type of Deposit: categories are 'No Deposit', 'Refundable', 'Non Refund'
> * agent = ID of travel agency who made booking
> * company= ID of company who made booking
> * days_in_waiting_list = Number of days booking on waiting before it is conform
> * customer_type = type of booking: categories are 'Transient', 'Contract', 'Transient-Party', 'Group'
> * adr = Average Daily Rate of hotel
> * required_car_parking_spaces = Number of car parking space required by guest
> * total_of_special_requests = Number of special request made by guest
> * reservation_status = Reservation status: categories are 'Check-Out', 'Canceled', 'No-Show'
> * reservation_status_date = date of reservation

<h2> :book: Exploratory Data Analysis</h2>

It is the approch to analyse the data using visualization techniques, it is used to discover trends,patterns or check the assumptions with the help of statistical summary.

> * most Of Booking is from City Hotel so, city Hotel Has 66% of share and rest 34% share is Resort Hotel.
> * August Month has nearly 14000 bookings and July month has more than 12000 bookings, these both are more preferred by people.
> * City hotel has 41% of Booking Cancellation while Resort hotel has 27% in Resort hotel.
> * Average Daily Rate is in between 50 to 150.
> * Lead time affects the Average Daily Rate as Lead Time Increase Average Daily Rate is Decreases.
> * Lead time is increases Cancellation of Booking is also Increases.
